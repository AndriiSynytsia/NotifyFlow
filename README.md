# NotifyFlow

Notification & Event Processing Platform built with Java and Spring Boot.

NotifyFlow is an open-source backend project focused on building a scalable and maintainable platform for processing notifications, events, retries, scheduling, and asynchronous workflows.

---

## Project Goals

- Build a production-style backend platform
- Practice scalable architecture patterns
- Explore event-driven workflows
- Implement reliable notification processing
- Create an open-source project for contributors
- Document architectural decisions and evolution

---

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA

### Database
- PostgreSQL

### Infrastructure
- Docker
